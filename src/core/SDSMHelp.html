Quality Control

Identifies data file size, missing data codes, and calculates the mean, maximum and minimum values in the file.

Transform Data

Transforms single or multi-column data files using standard functions (e.g., natural logs, squares, lags, binomials, etc.). Also extracts ensemble members from a multi-column data file.

Screen Variables

Identifies significant correlations between specified local weather variables and regional-scale atmospheric predictor variables. Also provides scatterplots for seasonal sub-sets.

Calibrate Model

Constructs monthly, seasonal or annual statistical downscaling models given daily weather data and regional-scale atmospheric predictor variables.

Weather Generator

Generates ensembles of synthetic weather series from calibrated statistical downscaling models given daily atmospheric predictor variables (observed data).

Summary Statistics

Computes a range of statistics (e.g., mean, variance, peaks over threshold, spell lengths, percentiles, etc.) given synthetic daily weather series produced by the Weather Generator or Generate Scenario operations.

Frequency Analysis

Computes and displays in graphical form a number of statistics for both observed and modelled data. The analysis includes the fitting of extreme value statistics, quantile-quantile plots, PDF plots and standard line plots.

Scenario Generator

Applies various treatments to weather data so that consequences and decisions can be evaluated.

Compare Results

Compares summary statistics produced by the Analyse operations using bar chart or line chart plots.

Time Series Analysis

Plots various times series data from modeled or observed data files.

Settings

Makes global changes to the system set-up, data handling, and model preferences.
